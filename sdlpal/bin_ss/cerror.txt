本體 RAM 的儲存尚未準備好。請至 SEGASATURN 本體的資料保存管理畫面把所有的記錄清除。(請作初期化)。押著  L  鍵或  R  鍵不放時再押 RESET 鍵，可回到 SEGASATURN 本體的資料保存管理畫面。卡匣 RAM 的儲存尚未準備好。請至 SEGASATURN 本體的資料保存管理畫面把所有的記錄清除。(請作初期化)。押著  L  鍵或  R  鍵不放時再押 RESET 鍵可回到 SEGASATURN 本體的資料保存管理畫面。在本體 RAM 的 SAVE DATA 無法使用。照此現狀開始遊戲的話，將無法保存記錄。如果還是要繼續遊戲，請押 START 鍵。這個遊戲必須空出     記憶容量。在 SEGASATURN 本體的資料保存管理畫面中，把其它遊戲記錄消除或是把遊戲 COPY 至卡匣 RAM中然後再開始遊戲。在卡匣 RAM 的 SAVE DATA 無法使用。
