You need the original full game or shareware data to run:
create "titus" dir in "home" folder (~/ or /usr/local/home/)
and put inside the game files.

START= begin (in menu) - pause/play (in game)
SELECT = exit from game (come back to Gmenu2X)
A = up/jump (as alternative to d-pad)
B = pickup/throw
Y = see energy and level/lifes info
L = enable/disable sound
R = "kill" Titus
